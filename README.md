Portefaix
=========

## Description

Some docker containers.

## Deployment

* Build the container :

        $ docker build -t="<yourname>/<name>" .

* Once the container is available, push it to the [Docker Index](index.docker.io) :

        $ sudo docker login
        $ sudo docker push <yourname>/<name>
